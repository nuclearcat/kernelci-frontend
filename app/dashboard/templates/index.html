Welcome to the KernelCI web dashboard for the upstream Linux kernel.

You'll find here all the results for kernel builds and tests run natively by KernelCI on mainline, linux-next, stable and a variety of maintainer branches.

To find out more about the project, see the main kernelci.org website.
